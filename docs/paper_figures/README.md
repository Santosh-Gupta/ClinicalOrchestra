# Paper Figure Sources

Source files for figures referenced in `docs/paper_draft_20260620.md` and
`docs/paper_workshop_compact_20260623.md`.

- `figure1_three_stage_funnel.mmd`: Mermaid source for the three-stage evaluation funnel.
- `figure1_three_stage_funnel.svg`: venue-ready vector rendering of the three-stage funnel.
- `figure2_judge_variance_floor.csv`: data for the judge variance floor figure/table.
- `figure2_judge_variance_floor.svg`: venue-ready vector plot of the judge variance floor.

The SVG files are self-contained and can be embedded directly or converted by the venue build pipeline.
